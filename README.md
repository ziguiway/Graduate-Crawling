# Graduate-Crawling

> *从工位搬回书桌，从"打工人"降级成"读研崽"。褪去职场伪装，重新解锁"爬行"技能——毕竟步子迈大了容易扯到，慢慢来*
---

## 关于这个仓库

在职场摸爬滚打两年，被KPI按在地上摩擦过，也体会过"赚钱如吃土"的辛酸，最终痛定思痛：还是回学校"避避风头"，重启读书副本。

这意味着，我要面对这些改变：
- 深知赚钱有多难，所以读书不敢摸鱼，主打一个"惜时如金"，毕竟学费不是大风刮来的
- 职场练出的"代码手速"还在，但面对学术论文，脑子直接宕机——主打一个"手脑脱节"，得重新磨合
- 习惯了深夜下班躺平，如今要强迫自己清晨爬去图书馆，生物钟：我是谁？我在哪？我要干嘛？

为什么叫 **Crawling**？因为在职场里，我好歹能稳稳"走路"，业务难题能拿捏，需求交付能搞定；回到校园才发现，学术之路全是坑，只能像刚学步的娃，从头慢慢爬、慢慢踩坑。

但我始终相信——爬过的路不会白爬，每一步笨拙的前行，都是在为未来铺路；哪怕偶尔绕路，也是独属于我的"读研限定弯路"，主打一个体验感拉满。

## 开学前目标（2026.09）

工作两年留下的习惯，有好有坏，有些要捡起来，有些要彻底改掉——毕竟读研不是打工，不能再用"摸鱼式交付"糊弄自己：

- [ ] 提升英语论文阅读效率：从"逐字逐句抠+翻译软件续命"，进阶到"扫读抓大意"，争取摆脱翻译软件依赖
- [ ] 重拾数学基础：线代、概率早已还给大学老师，现在捡起来，主打一个"似曾相识，却认不出"
- [ ] 明确兴趣方向：不求精准定位，但要有大致靶子，避免盲目摸索——总不能读了三年，还不知道自己在学啥
- [ ] 调整心态：从职场"快速交付、追求结果"，切换到学术"长期深耕、接纳不完美"，戒掉"急功近利"的坏毛病
- [ ] 叩问初心：想清楚"我为什么回来读研？"——答案会慢慢沉淀，最终放在README末尾

## 工作两年教会我的事

这两年的职场经历，不是负担，是我读研路上的"隐藏buff"——毕竟踩过的坑，都能变成未来的底气：

1. **时间比想象中更宝贵** → 不再浪费光阴摸鱼，学会有计划地"爬行"，毕竟读研三年，浪费不起
2. **大部分问题都有标准答案** → 但学术没有，要学会与不确定共处，主打一个"佛系探索，随缘进步"
3. **累了就休息，不必硬撑** → 比起刚毕业时的逞强，如今更懂得劳逸结合——毕竟读研是马拉松，不是百米冲刺
4. **做事要有产出感** → 但学术的产出周期很长，要耐得住寂寞，接受"慢工出细活"
